Wanderloom

A web application for creating meaningful photo albums of travel memories.

  • Group Project: Wanderloom App

Wanderloom is a web application designed to help users create organized collections of their travel photos. The app allows users to upload photos, organize them neatly by geolocation into albums, and return to view their albums whenever they want, making it easy to relive travel experiences.

Technology:

React, Firebase, JavaScript, HTML, CSS

My Role:

In this project, I contributed to the project documentation as the documentation specialist, team scribe, and worked in tandem with my team to develop the application.

  • Managed documentation, including the weekly team report and retrospectives to ensure clear communication of project goals, features, and development progress.
  • Collaborated with team members to design and implement the user interface using React, ensuring a seamless and intuitive user experience.
  • Integrated Firebase Firestore for backend services, including uploading photo geolocation data to storage.
  • Participated in code reviews and contributed to debugging and testing efforts to maintain code quality and functionality.
